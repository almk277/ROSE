RMD stands for ROSE module, main ROSE binary format.

Module includes following sections:

* #head  - general module information and other sections sizes;
* #exp   - exported procedures;
* #ptbl  - every procedure location and description;
* #mtbl  - imported module list;
* #imp   - imported procedures;
* #text  - all instructions in all procedures;
* #sym   - all loadable symbols;
* #str   - constant array pool;
* #debug - debug information.

Section #debug might be ommited.

#head:
	* name   - #sym string with short module name;
	* time   - module creation date and time;
	* RMDver - RMD format version;
	* ver    - module version;
	* placement for all sections.

#exp:
	* name    - #sym string with procedure export name;
	* proto   - #sym string with prototype code.
	* idx     - index into #ptbl.

#ptbl:
	* addr   - offset in #text.

#mtbl:
	* name    - #sym string with module full name;
	* version - required module version.

#imp:
	* name   - #sym string with procedure name;
	* module - #mtbl index for module exporting this procedure;
	* slot   - index into procedure module #exp;
	* addr   - cached procedure offset.
addr and slot might be ommited.
